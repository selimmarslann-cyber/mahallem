## 🎨 TASARIM SİSTEMİ DETAYLARI

### Renk Paleti
```css
Background: #F5F5F7 (off-white)
Primary: #FF7A00 (premium orange)
Secondary: #0F172A, #1E293B (slate)
Success: #10B981 (green)
Error: #EF4444 (red)
Warning: #F59E0B (yellow)
```

### Typography
- Font: Inter (sans-serif)
- Headings: Bold, net, minimal shadow
- Body: Comfortable line-height (1.6-1.8)

### Kart Sistemi
- Border radius: 12-16px
- Shadow: Hafif, yumuşak
- Hover: Scale + shadow artışı
- İçerik: Esnaf adı, puan, fiyat, mesafe, kategori

### Responsive
- Mobile: 320-768px (bottom nav)
- Tablet: 768-1024px
- Desktop: 1024px+ (top nav)

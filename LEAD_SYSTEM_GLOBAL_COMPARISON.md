## 📊 EXECUTIVE SUMMARY

### Mevcut Sistem Özeti
Mahallem platformu, Türkiye pazarı için özel olarak tasarlanmış bir hizmet eşleştirme platformudur. Yeni eklenen **Lead Sistemi**, müşterilerin ücretsiz ilan açmasına ve ustaların iletişim bilgilerine erişmek için lead başına ödeme yapmasına dayanmaktadır.

### Temel Özellikler
- ✅ **AI-Powered Kategori Tespiti**: Groq AI ile otomatik kategori seçimi
- ✅ **Level-Based Pricing**: 1-10 arası level sistemi, 20-90 TL arası sabit ücret
- ✅ **Wallet Integration**: Cüzdan sistemi ile ödeme akışı
- ✅ **Contact Gating**: İletişim bilgileri ödeme sonrası açılıyor
- ✅ **Category-Based Notifications**: Ana kategoriye göre bildirim sistemi

#### A. Garanti Sistemi
```typescript
// Önerilen: Lead iade sistemi
interface LeadRefundPolicy {
  refundConditions: {
    noContact: boolean      // Usta 24 saat içinde iletişime geçmezse
    customerCancelled: boolean // Müşteri iptal ederse
    duplicateLead: boolean    // Aynı usta aynı lead'i iki kez alırsa
  }
  refundAmount: number      // %100 veya kısmi
  processingTime: number    // İade süresi (gün)
}
```

- Lead satın alındıktan sonra 24 saat içinde usta iletişime geçmezse otomatik iade
